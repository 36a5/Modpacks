# d7oom's-tool Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** An OP-only wooden-axe wand for `pack-two` that selects a region with two right-clicks, previews the result as translucent ghost blocks, then fills / moves / clones / exports it.

**Architecture:** WorldEdit 7.2.15 (Forge) is the edit engine — it owns `//set`, `//copy`, `//paste`, `//schem`, `//undo`, and a block argument that already tab-completes every block in the pack. A new mod, `d7oomtool`, supplies only what WorldEdit cannot: the item, the right-click/right-click selection, the client-side ghost renderer, drag-move/clone to an arbitrary point, and vanilla `.nbt` export. The two talk through command dispatch (`//pos1` / `//pos2`) plus a `compileOnly` dependency on `worldedit-core` for clipboard pastes at exact coordinates.

**Tech Stack:** Java 17, Minecraft 1.20.1, Forge 47.4.18, ForgeGradle 6, `worldedit-core:7.2.15` (compileOnly), JUnit 5, Forge GameTest, packwiz.

**Spec:** [docs/superpowers/specs/2026-07-13-d7oom-tool-design.md](../specs/2026-07-13-d7oom-tool-design.md)

## Global Constraints

- **Minecraft 1.20.1 / Forge 47.4.18 / Java 17.** These are fixed by `pack-two/pack.toml`. Do not bump any of them.
- **Mod ID is `d7oomtool`. Item ID is `d7oomtool:d7oom_axe`.** The registry ID cannot contain an apostrophe; the display name `d7oom's-tool` lives in the lang file only.
- **Mod source lives at `mods/d7oomtool/` in the repo root — NOT inside `pack-two/`.** CI indexes every file under a pack folder and fails the build if an indexed file isn't committed. A Gradle project inside `pack-two/` would be indexed and would break every player's install.
- **WorldEdit comes from CurseMaven — `fg.deobf("curse.maven:worldedit-225608:4586218")`.** That is byte-for-byte the jar the pack ships, so compile-time classes and runtime classes cannot drift. Do NOT use `maven.enginehub.org`'s separately-published `worldedit-core`. ForgeGradle does not shade, so nothing is bundled into our jar.
- **WorldEdit is a `mandatory = true` dependency in `mods.toml`.** "WorldEdit is missing" is therefore not a runtime state — Forge refuses to launch and says why. Write no `isPresent()` check, no chat-error fallback, no defensive branch. A path that can never be reached is just untested code.
- **All WorldEdit access goes through `WorldEditBridge`, and it uses the Java API, not command strings.**
- **Every click handler and every `/d7` subcommand independently re-checks `hasPermissions(opLevel)`.** Acquisition being OP-gated is not a substitute.
- **Preview volume cap is `previewMaxVolume`, default `32768`.** Above it, the preview turns OFF entirely (no fallback wireframe ghost). The edit itself stays uncapped.
- **Block-entity NBT always travels on a move/clone. Entities travel only with `-e`.**
- **The local packwiz binary is `c:\Minecraft-dev-workspace\tools\packwiz.exe`** (it is not on PATH). CI installs its own via Go.
- **Git remote is `https://github.com/36a5/Modpacks`.** Release asset URLs derive from this.
- Branch: `feature/d7oom-tool`. Commit after every task.

```java
    @Override
    public boolean canAttackBlock(BlockState state, Level level, BlockPos pos, Player player) {
        return false;
```

```java
import net.minecraft.core.BlockPos;
import net.minecraft.core.Vec3i;
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class RegionTest {
    @Test
    void normalizesCornersRegardlessOfClickOrder() {
        Region a = Region.of(new BlockPos(10, 70, 10), new BlockPos(0, 60, 0));
        Region b = Region.of(new BlockPos(0, 60, 0), new BlockPos(10, 70, 10));
        assertEquals(new BlockPos(0, 60, 0), a.min());
        assertEquals(new BlockPos(10, 70, 10), a.max());
        assertEquals(a, b, "clicking the far corner first must produce the same region");
    }

    @Test
    void volumeIsInclusiveOfBothCorners() {
        // 0..0 on every axis is one block, not zero.
        assertEquals(1L, Region.of(BlockPos.ZERO, BlockPos.ZERO).volume());
        // 0..1 on every axis is 2*2*2.
        assertEquals(8L, Region.of(BlockPos.ZERO, new BlockPos(1, 1, 1)).volume());
        assertEquals(new Vec3i(2, 2, 2), Region.of(BlockPos.ZERO, new BlockPos(1, 1, 1)).size());
    }

    @Test
    void volumeDoesNotOverflowInt() {
        // 2000^3 = 8e9, which is > Integer.MAX_VALUE. The cap check must not wrap negative.
        Region r = Region.of(BlockPos.ZERO, new BlockPos(1999, 1999, 1999));
        assertEquals(8_000_000_000L, r.volume());
        assertTrue(r.volume() > 0, "volume must not overflow into a negative number");
    }

    @Test
    void containsIsInclusive() {
        Region r = Region.of(BlockPos.ZERO, new BlockPos(2, 2, 2));
        assertTrue(r.contains(BlockPos.ZERO));
        assertTrue(r.contains(new BlockPos(2, 2, 2)));
        assertFalse(r.contains(new BlockPos(3, 0, 0)));
    }

    @Test
    void offsetMovesBothCorners() {
        Region r = Region.of(BlockPos.ZERO, new BlockPos(1, 1, 1)).offset(new Vec3i(5, 0, -5));
        assertEquals(new BlockPos(5, 0, -5), r.min());
        assertEquals(new BlockPos(6, 1, -4), r.max());
    }
}
```

```java
package com.d7oom.d7oomtool.selection;

import net.minecraft.core.BlockPos;
import net.minecraft.core.Vec3i;

/** An inclusive cuboid. Corners are normalized on construction, so click order never matters. */
public record Region(BlockPos min, BlockPos max) {

    public static Region of(BlockPos a, BlockPos b) {
        return new Region(
                new BlockPos(Math.min(a.getX(), b.getX()), Math.min(a.getY(), b.getY()), Math.min(a.getZ(), b.getZ())),
                new BlockPos(Math.max(a.getX(), b.getX()), Math.max(a.getY(), b.getY()), Math.max(a.getZ(), b.getZ())));
    }

    public Vec3i size() {
        return new Vec3i(max.getX() - min.getX() + 1, max.getY() - min.getY() + 1, max.getZ() - min.getZ() + 1);
    }

    /** long, not int: a 2000-block cube is 8e9 and would wrap negative as an int, defeating the cap check. */
    public long volume() {
        Vec3i s = size();
        return (long) s.getX() * (long) s.getY() * (long) s.getZ();
    }

    public boolean contains(BlockPos p) {
        return p.getX() >= min.getX() && p.getX() <= max.getX()
            && p.getY() >= min.getY() && p.getY() <= max.getY()
            && p.getZ() >= min.getZ() && p.getZ() <= max.getZ();
    }

    public Region offset(Vec3i d) {
        return new Region(min.offset(d), max.offset(d));
    }
}
```

```java
package com.d7oom.d7oomtool.selection;

import net.minecraft.core.BlockPos;
import net.minecraft.world.level.block.Block;
import net.minecraft.world.level.block.Blocks;
import javax.annotation.Nullable;

/** One player's selection state. Server-authoritative; the client holds a read-only mirror. */
public class Selection {
    @Nullable private BlockPos pos1;
    @Nullable private BlockPos pos2;
    private Mode mode = Mode.IDLE;
    private Block material = Blocks.STONE;

    /** First call sets pos1, second sets pos2, third starts over. */
    public void setCorner(BlockPos pos) {
        if (pos1 == null || pos2 != null) {
            pos1 = pos;
            pos2 = null;
            mode = Mode.SELECTING;
        } else {
            pos2 = pos;
        }
    }

    public void setPos1(BlockPos pos) { this.pos1 = pos; this.pos2 = null; this.mode = Mode.SELECTING; }

    @Nullable public BlockPos pos1() { return pos1; }
    @Nullable public BlockPos pos2() { return pos2; }

    @Nullable public Region region() {
        return (pos1 == null || pos2 == null) ? null : Region.of(pos1, pos2);
    }

    public Mode mode() { return mode; }
```

```java
package com.d7oom.d7oomtool;

import com.d7oom.d7oomtool.selection.Region;
import com.d7oom.d7oomtool.selection.Selection;
import com.d7oom.d7oomtool.selection.SelectionStore;
import net.minecraft.core.BlockPos;
import net.minecraft.network.chat.Component;
import net.minecraft.server.level.ServerPlayer;

/** Every entry point re-checks permission. Acquisition being OP-gated is not a substitute. */
public final class Interactions {
    private Interactions() {}

    public static boolean mayUse(ServerPlayer player) {
        return player.hasPermissions(Config.opLevel());
    }

    public static void setCorner(ServerPlayer player, BlockPos pos) {
        if (!mayUse(player)) {
            return; // Silently inert. A non-OP is holding a decorative axe.
        }
        if (!withinReach(player, pos)) {
            player.sendSystemMessage(Component.translatable("d7oomtool.msg.too_far", Config.maxReach()));
            return;
        }
        Selection sel = SelectionStore.SERVER.get(player.getUUID());
        sel.setCorner(pos);

        Region region = sel.region();
        if (region == null) {
            player.sendSystemMessage(Component.translatable(
                    "d7oomtool.msg.pos1", pos.getX(), pos.getY(), pos.getZ()));
        } else {
            player.sendSystemMessage(Component.translatable(
                    "d7oomtool.msg.pos2", pos.getX(), pos.getY(), pos.getZ(), region.volume()));
        }
        // Task 4 syncs to the client here; Task 7 pushes into WorldEdit here.
    }

    public static void clear(ServerPlayer player) {
        if (!mayUse(player)) return;
        SelectionStore.SERVER.clear(player.getUUID());
        player.sendSystemMessage(Component.translatable("d7oomtool.msg.cleared"));
    }

    static boolean withinReach(ServerPlayer player, BlockPos pos) {
        int max = Config.maxReach();
        return pos.distSqr(player.blockPosition()) <= (long) max * max;
    }
}
```

```java
package com.d7oom.d7oomtool.net;

import com.d7oom.d7oomtool.selection.Mode;
import com.d7oom.d7oomtool.selection.Selection;
import net.minecraft.core.BlockPos;
import net.minecraft.network.FriendlyByteBuf;
import net.minecraft.resources.ResourceLocation;
import net.minecraft.world.level.block.Block;
import net.minecraft.world.level.block.Blocks;
import net.minecraftforge.api.distmarker.Dist;
import net.minecraftforge.fml.DistExecutor;
import net.minecraftforge.network.NetworkEvent;
import net.minecraftforge.registries.ForgeRegistries;

import javax.annotation.Nullable;
import java.util.function.Supplier;

public record SelectionSyncPacket(@Nullable BlockPos pos1, @Nullable BlockPos pos2, Mode mode, Block material) {

    public static SelectionSyncPacket of(Selection s) {
        return new SelectionSyncPacket(s.pos1(), s.pos2(), s.mode(), s.material());
    }

    public static void encode(SelectionSyncPacket p, FriendlyByteBuf buf) {
        buf.writeBoolean(p.pos1 != null);
        if (p.pos1 != null) buf.writeBlockPos(p.pos1);
        buf.writeBoolean(p.pos2 != null);
        if (p.pos2 != null) buf.writeBlockPos(p.pos2);
        buf.writeEnum(p.mode);
        buf.writeResourceLocation(ForgeRegistries.BLOCKS.getKey(p.material));
    }

    public static SelectionSyncPacket decode(FriendlyByteBuf buf) {
        BlockPos p1 = buf.readBoolean() ? buf.readBlockPos() : null;
        BlockPos p2 = buf.readBoolean() ? buf.readBlockPos() : null;
        Mode mode = buf.readEnum(Mode.class);
        ResourceLocation matId = buf.readResourceLocation();
        Block mat = ForgeRegistries.BLOCKS.getValue(matId);
        return new SelectionSyncPacket(p1, p2, mode, mat == null ? Blocks.STONE : mat);
    }

    public static void handle(SelectionSyncPacket p, Supplier<NetworkEvent.Context> ctx) {
        ctx.get().enqueueWork(() -> DistExecutor.unsafeRunWhenOn(Dist.CLIENT,
                () -> () -> com.d7oom.d7oomtool.client.ClientSelection.accept(p)));
        ctx.get().setPacketHandled(true);
    }
}
```

## Task 6: The ghost renderer

**Files:**
- Create: `src/client/java/com/d7oom/d7oomtool/client/GhostLevel.java`
- Create: `src/client/java/com/d7oom/d7oomtool/client/GhostRenderer.java`
- Modify: `client/OutlineRenderer.java` (call the ghost renderer from the same event)

**Interfaces:**
- Consumes: `ClientSelection.previewRegion()`, `ClientSelection.material`, `Config.previewMaxVolume()`.
- Produces: `GhostRenderer.render(PoseStack, Matrix4f, Vec3, Region, BlockState)`, `GhostRenderer.invalidate()`.

**The two ideas that make this work:**

1. **`GhostLevel` is a `BlockAndTintGetter` that lies.** It delegates to the real level except inside the region, where it returns the ghost material. Because Minecraft's block renderer culls a face when the neighbour is solid, feeding it a level where the whole region is already filled means **interior faces are never generated at all**. Without this, a 32³ region emits ~200k faces instead of ~6k, and translucent interior faces would double-darken into mush.

2. **Cache the geometry in a `VertexBuffer`.** Rebuild only when the region or material changes — never per frame. After the rebuild, panning the camera costs one draw call.

```java
import com.d7oom.d7oomtool.selection.Region;
import net.minecraft.core.BlockPos;
import net.minecraft.world.level.BlockAndTintGetter;
import net.minecraft.world.level.ColorResolver;
import net.minecraft.world.level.Level;
import net.minecraft.world.level.LightLayer;
import net.minecraft.world.level.block.entity.BlockEntity;
import net.minecraft.world.level.block.state.BlockState;
import net.minecraft.world.level.material.FluidState;
import javax.annotation.Nullable;

/** The real level, except every block inside the region reads back as the ghost material. */
public record GhostLevel(Level real, Region region, BlockState ghost) implements BlockAndTintGetter {

    @Override
    public BlockState getBlockState(BlockPos pos) {
        return region.contains(pos) ? ghost : real.getBlockState(pos);
    }

    @Override
    public FluidState getFluidState(BlockPos pos) {
        return region.contains(pos) ? ghost.getFluidState() : real.getFluidState(pos);

    @Nullable
    @Override
    public BlockEntity getBlockEntity(BlockPos pos) {
        return region.contains(pos) ? null : real.getBlockEntity(pos);

    // a dark region invisible, which defeats the purpose of a preview.
    @Override public float getShade(net.minecraft.core.Direction dir, boolean shade) { return real.getShade(dir, shade); }
    @Override public net.minecraft.world.level.lighting.LevelLightEngine getLightEngine() { return real.getLightEngine(); }
    @Override public int getBrightness(LightLayer layer, BlockPos pos) { return 15; }
    @Override public int getRawBrightness(BlockPos pos, int amount) { return 15; }
    @Override public int getBlockTint(BlockPos pos, ColorResolver resolver) { return real.getBlockTint(pos, resolver); }
    @Override public int getHeight() { return real.getHeight(); }
    @Override public int getMinBuildHeight() { return real.getMinBuildHeight(); }
```

```java
import net.minecraft.util.RandomSource;
import net.minecraft.world.inventory.InventoryMenu;
import net.minecraft.world.level.block.state.BlockState;
import net.minecraft.world.phys.Vec3;
import net.minecraftforge.client.model.data.ModelData;
import org.joml.Matrix4f;

/** Builds the ghost geometry once per region/material change and redraws it in one call. */
public final class GhostRenderer {

    @org.jetbrains.annotations.Nullable private static VertexBuffer buffer;
    @org.jetbrains.annotations.Nullable private static Region builtRegion;
    @org.jetbrains.annotations.Nullable private static BlockState builtState;
    private static BlockPos origin = BlockPos.ZERO;

    private GhostRenderer() {}

    public static void invalidate() {
        builtRegion = null;
        builtState = null;
    }

    public static void render(PoseStack pose, Matrix4f projection, Vec3 cam, Region region, BlockState state) {
        if (!region.equals(builtRegion) || !state.equals(builtState)) {
            rebuild(region, state);
        }
        if (buffer == null) return;

        pose.pushPose();
        pose.translate(origin.getX() - cam.x, origin.getY() - cam.y, origin.getZ() - cam.z);

        RenderSystem.enableBlend();
        RenderSystem.defaultBlendFunc();
        RenderSystem.depthMask(false); // translucent ghosts must not occlude each other
        RenderSystem.setShader(GameRenderer::getRendertypeTranslucentShader);
        RenderSystem.setShaderTexture(0, InventoryMenu.BLOCK_ATLAS);
        RenderSystem.setShaderColor(1.0f, 1.0f, 1.0f, 0.45f);
        Minecraft.getInstance().gameRenderer.lightTexture().turnOnLightLayer();

        buffer.bind();
        buffer.drawWithShader(pose.last().pose(), projection, RenderSystem.getShader());
        VertexBuffer.unbind();

        Minecraft.getInstance().gameRenderer.lightTexture().turnOffLightLayer();
        RenderSystem.setShaderColor(1.0f, 1.0f, 1.0f, 1.0f);
        RenderSystem.depthMask(true);
        RenderSystem.disableBlend();

        pose.popPose();
    }

    private static void rebuild(Region region, BlockState state) {
        Minecraft mc = Minecraft.getInstance();
        if (mc.level == null) return;

        origin = region.min();
        // The lying level: inside the region, every block already reads as the ghost material,
        // so the block renderer culls every interior face for us.
        GhostLevel ghostLevel = new GhostLevel(mc.level, region, state);

        BufferBuilder builder = new BufferBuilder(RenderType.translucent().bufferSize());
        builder.begin(VertexFormat.Mode.QUADS, DefaultVertexFormat.BLOCK);

        PoseStack pose = new PoseStack();
        RandomSource random = RandomSource.create();
        BlockPos.MutableBlockPos pos = new BlockPos.MutableBlockPos();

        for (int x = region.min().getX(); x <= region.max().getX(); x++) {
            for (int y = region.min().getY(); y <= region.max().getY(); y++) {
                for (int z = region.min().getZ(); z <= region.max().getZ(); z++) {
                    pos.set(x, y, z);
                    pose.pushPose();
                    pose.translate(x - origin.getX(), y - origin.getY(), z - origin.getZ());
                    mc.getBlockRenderer().renderBatched(
                            state, pos, ghostLevel, pose, builder,
                            true,           // checkSides — this is what culls interior faces
                            random, ModelData.EMPTY, RenderType.translucent());
                    pose.popPose();
                }
            }
        }

        BufferBuilder.RenderedBuffer rendered = builder.end();
        if (buffer == null) buffer = new VertexBuffer(VertexBuffer.Usage.STATIC);
        buffer.bind();
        buffer.upload(rendered);
        VertexBuffer.unbind();

        builtRegion = region;
        builtState = state;
    }
}
```

```java
package com.d7oom.d7oomtool.we;

import com.d7oom.d7oomtool.selection.Region;
import net.minecraft.commands.CommandSourceStack;
import net.minecraft.core.BlockPos;
import net.minecraft.network.chat.Component;
import net.minecraft.server.level.ServerPlayer;

/**
 * The ONLY class that touches WorldEdit. Everything here degrades to a chat message when
 * WorldEdit is missing — a missing dependency must never crash the server.
 */
public final class WorldEditBridge {

    private static Boolean present;

    private WorldEditBridge() {}

    public static boolean isPresent() {
        if (present == null) {
            try {
                Class.forName("com.sk89q.worldedit.WorldEdit");
                present = Boolean.TRUE;
            } catch (Throwable t) {
                present = Boolean.FALSE;
            }
        }
        return present;
    }

    public static boolean requireWorldEdit(ServerPlayer player) {
        if (isPresent()) return true;
        player.sendSystemMessage(Component.translatable("d7oomtool.msg.no_worldedit"));
        return false;
    }

    /**
     * Hands our selection to WorldEdit by running its own commands as the player.
     * Dispatch, not API: nothing here breaks at compile time if WorldEdit changes internals.
     *
     * WorldEdit registers its commands with a literal leading slash ("/pos1"), so the string
     * we pass must start with "//pos1" — performPrefixedCommand strips exactly one slash.
     */
    public static void pushSelection(ServerPlayer player, Region region) {
        if (!isPresent()) return;
        CommandSourceStack src = player.createCommandSourceStack()
                .withPermission(4)
                .withSuppressedOutput(); // WorldEdit's own "First position set" spam is redundant with ours
        BlockPos a = region.min();
        BlockPos b = region.max();
        player.server.getCommands().performPrefixedCommand(src,
                "//pos1 " + a.getX() + "," + a.getY() + "," + a.getZ());
        player.server.getCommands().performPrefixedCommand(src,
                "//pos2 " + b.getX() + "," + b.getY() + "," + b.getZ());
    }
}
```

```java
    public static void onRegisterCommands(net.minecraftforge.event.RegisterCommandsEvent event) {
        D7Command.register(event.getDispatcher(), event.getBuildContext());

    /**
     * The fill block isn't known until the player types //set. So remember the last one they used

    @SubscribeEvent
    public static void onCommand(net.minecraftforge.event.CommandEvent event) {
        var source = event.getParseResults().getContext().getSource();
        if (!(source.getEntity() instanceof ServerPlayer player)) return;

        String raw = event.getParseResults().getReader().getString(); // e.g. "//set create:andesite_casing"
        String prefix = "//set ";
        if (!raw.startsWith(prefix)) return;

        String arg = raw.substring(prefix.length()).trim();
        int cut = arg.indexOf('[');            // strip a blockstate:  stone[foo=bar]
        if (cut > 0) arg = arg.substring(0, cut);
        cut = arg.indexOf(' ');
        if (cut > 0) arg = arg.substring(0, cut);

        net.minecraft.resources.ResourceLocation id = net.minecraft.resources.ResourceLocation.tryParse(arg);
        if (id == null) return;
        var block = net.minecraftforge.registries.ForgeRegistries.BLOCKS.getValue(id);
        if (block == null) return;

        SelectionStore.SERVER.get(player.getUUID()).setMaterial(block);
        Net.syncTo(player);
```

```java
    /**
     * Copy (and optionally cut) a region and paste it so that its minimum corner lands on destMin.
     *
     * Cut and paste happen in ONE EditSession, so //undo reverses the whole move in a single step
     * instead of leaving a hole where the source used to be.
     *
     * The clipboard is filled before the cut, so an overlapping source and destination is safe.
     */
    public static boolean moveOrClone(ServerPlayer player, Region source, BlockPos destMin,
                                      boolean cut, boolean entities) {
        if (!isPresent()) return false;
        try {
            com.sk89q.worldedit.entity.Player wePlayer = actorFor(player);
            if (wePlayer == null) return false;

            com.sk89q.worldedit.world.World weWorld = wePlayer.getWorld();
            com.sk89q.worldedit.LocalSession session =
                    com.sk89q.worldedit.WorldEdit.getInstance().getSessionManager().get(wePlayer);

            com.sk89q.worldedit.math.BlockVector3 min = vec(source.min());
            com.sk89q.worldedit.math.BlockVector3 max = vec(source.max());
            com.sk89q.worldedit.regions.CuboidRegion region =
                    new com.sk89q.worldedit.regions.CuboidRegion(weWorld, min, max);

            try (com.sk89q.worldedit.EditSession edit = com.sk89q.worldedit.WorldEdit.getInstance()
                    .newEditSessionBuilder().world(weWorld).actor(wePlayer).maxBlocks(-1).build()) {

                com.sk89q.worldedit.extent.clipboard.BlockArrayClipboard clipboard =
                        new com.sk89q.worldedit.extent.clipboard.BlockArrayClipboard(region);
                clipboard.setOrigin(min);

                com.sk89q.worldedit.function.operation.ForwardExtentCopy copy =
                        new com.sk89q.worldedit.function.operation.ForwardExtentCopy(edit, region, clipboard, min);
                copy.setCopyingEntities(entities);
                com.sk89q.worldedit.function.operation.Operations.complete(copy);

                if (cut) {
                    edit.setBlocks((com.sk89q.worldedit.regions.Region) region,
                            com.sk89q.worldedit.world.block.BlockTypes.AIR.getDefaultState());
                }

                com.sk89q.worldedit.function.operation.Operation paste =
                        new com.sk89q.worldedit.session.ClipboardHolder(clipboard)
                                .createPaste(edit)
                                .to(vec(destMin))
                                .copyEntities(entities)
                                .ignoreAirBlocks(false)
                                .build();
                com.sk89q.worldedit.function.operation.Operations.complete(paste);

                // Hand the session to WorldEdit's history so //undo works on our edits too.
                session.remember(edit);
            }
            return true;
        } catch (Throwable t) {
            player.sendSystemMessage(net.minecraft.network.chat.Component.literal(
                    "d7oom's-tool: the edit failed — " + t.getMessage()));
            return false;
        }
    }

    private static com.sk89q.worldedit.math.BlockVector3 vec(BlockPos p) {
        return com.sk89q.worldedit.math.BlockVector3.at(p.getX(), p.getY(), p.getZ());
    }

    /**
     * Resolve the WorldEdit Player for a Minecraft player by UUID.
     *
     * Deliberately does NOT use ForgeAdapter: the worldedit-forge artifact is not published to
     * Maven, only worldedit-core is. getConnectedUsers() is on the platform-agnostic API.
     */
    @javax.annotation.Nullable
    private static com.sk89q.worldedit.entity.Player actorFor(ServerPlayer player) {
        com.sk89q.worldedit.extension.platform.Platform platform =
                com.sk89q.worldedit.WorldEdit.getInstance().getPlatformManager()
                        .queryCapability(com.sk89q.worldedit.extension.platform.Capability.GAME_HOOKS);
        for (com.sk89q.worldedit.extension.platform.Actor actor : platform.getConnectedUsers()) {
            if (actor instanceof com.sk89q.worldedit.entity.Player p
                    && p.getUniqueId().equals(player.getUUID())) {
                return p;
            }
        }
        return null;
    }
```

```java
package com.d7oom.d7oomtool.net;

import com.d7oom.d7oomtool.Config;
import com.d7oom.d7oomtool.Interactions;
import com.d7oom.d7oomtool.selection.Mode;

import net.minecraft.core.BlockPos;
import net.minecraft.network.FriendlyByteBuf;
import net.minecraft.network.chat.Component;
import net.minecraft.server.level.ServerPlayer;
import net.minecraftforge.network.NetworkDirection;
import net.minecraftforge.network.NetworkEvent;
import java.util.function.Supplier;

/** The block the client's ghost is anchored at, sent on a right-click while in MOVE/CLONE mode. */
public record CommitPacket(BlockPos anchor) {

    static void register(int id) {
        Net.CHANNEL.messageBuilder(CommitPacket.class, id, NetworkDirection.PLAY_TO_SERVER)
                .encoder((p, buf) -> buf.writeBlockPos(p.anchor))
                .decoder(buf -> new CommitPacket(buf.readBlockPos()))
                .consumerMainThread(CommitPacket::handle)
                .add();
    }

    private static void handle(CommitPacket p, Supplier<NetworkEvent.Context> ctx) {
        ctx.get().enqueueWork(() -> {
            ServerPlayer player = ctx.get().getSender();
            if (player == null) return;

            // Validate everything. The client sent this; it is not to be trusted.
            if (!Interactions.mayUse(player)) return;
            if (!Interactions.withinReach(player, p.anchor)) {
                player.sendSystemMessage(Component.translatable("d7oomtool.msg.too_far", Config.maxReach()));
                return;
            }
```

```java
package com.d7oom.d7oomtool.structure;

import com.d7oom.d7oomtool.D7oomTool;
import com.d7oom.d7oomtool.selection.Region;
import net.minecraft.core.Vec3i;
import net.minecraft.nbt.CompoundTag;
import net.minecraft.nbt.NbtIo;
import net.minecraft.server.level.ServerLevel;
import net.minecraft.world.level.levelgen.structure.templatesystem.StructureTemplate;
import net.minecraft.world.level.storage.LevelResource;

import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.regex.Pattern;

/** Writes a selection as a vanilla structure .nbt, pasteable with /place template. */
public final class NbtExporter {

    /** The name becomes a filename. Anything outside this alphabet is an arbitrary file write. */
    private static final Pattern SAFE_NAME = Pattern.compile("^[a-z0-9_-]{1,64}$");

    private NbtExporter() {}

    public static boolean isValidName(String name) {
        return name != null && SAFE_NAME.matcher(name).matches();
    }

    /**
     * Writes to world/generated/d7oomtool/structures/&lt;name&gt;.nbt — exactly where
     * /place template d7oomtool:&lt;name&gt; reads from, so a saved structure pastes back
     * with no mod involved at all.
     */
    public static Path export(ServerLevel level, Region region, String name, boolean entities) throws IOException {
        if (!isValidName(name)) {
            throw new IllegalArgumentException("unsafe structure name: " + name);
        }
        Vec3i size = region.size();

        StructureTemplate template = new StructureTemplate();
        template.fillFromWorld(level, region.min(), size, entities, null);
        CompoundTag tag = template.save(new CompoundTag());

        Path dir = level.getServer().getWorldPath(LevelResource.GENERATED_DIR)
                .resolve(D7oomTool.MODID)
                .resolve("structures");
        Files.createDirectories(dir);

        Path file = dir.resolve(name + ".nbt");
        NbtIo.writeCompressed(tag, file.toFile());
        return file;
    }
}
```

```markdown
---

## d7oom's-tool — the build wand

An OP-only wand for `pack-two`. WorldEdit does the edits; `d7oomtool` supplies the item, the
ghost preview, drag-move/clone, and vanilla `.nbt` export.

```
/give <player> d7oomtool:d7oom_axe
```

`/give` is already permission level 2, and every click and `/d7` command re-checks level 2 anyway.
There is no recipe and no loot-table entry — a command is the only way to get one.

### Using it

| Input | Effect |
|---|---|
| Right-click | Set a corner. First click = pos1, second = pos2, third starts over. |
| Left-click a block | Set pos1. |
| Shift + right-click | Clear the selection. |

Between the two clicks a translucent ghost of the region tracks your crosshair. Once both corners
are down, the selection is handed to WorldEdit, so **every WorldEdit command works on it**:

```
//set create:andesite_casing      //replace stone deepslate
//copy   //paste   //undo   //schem save mybase
```

The ghost previews in the last block you `//set` (stone until you've used one). `/d7 mat <block>`
overrides it.

### `/d7`

| Command | What it does |
|---|---|
| `/d7 move [-e]` | The selection's real blocks ghost at your crosshair; right-click drops them there. The source is left as air. |
| `/d7 clone [-e]` | Same, but the original stays. |
| `/d7 mat <block>` | Sets the preview material. |
| `/d7 nbt <name> [-e]` | Saves the selection to `world/generated/d7oomtool/structures/<name>.nbt`. Paste it back with `/place template d7oomtool:<name>` — no mod needed. |
| `/d7 cancel` | Clears the selection. |

`-e` also moves entities (mobs, item frames, armor stands). **Without it, entities stay put — but
block-entity NBT always travels**, so chest contents, sign text and Create machine state survive a
move either way.

`//undo` reverses a `/d7 move` in one step, hole and all.

### Gotchas

- **Regions above 32,768 blocks (32³) do not preview.** The wireframe box and the volume readout
  stay; the ghost blocks turn off. The edit itself is uncapped — this is a client framerate limit,
  not an edit limit. Tune with `previewMaxVolume` in `config/d7oomtool-common.toml`.
- A region crossing an unloaded chunk is **refused**, not half-applied.
- WorldEdit 7.2.15 is the last Forge build EngineHub ever shipped. There is no upgrade to chase.
```
