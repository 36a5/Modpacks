# al Shabab — TLauncher Setup (non-premium players)

## One-time setup

1. Install Java 17 (Temurin): https://adoptium.net
2. In TLauncher, select version **Forge 1.20.1** (47.4.x) and press *Install*. Launch it
   once so it creates the game folder, then close the game.
3. Download `update.bat` from this folder and run it. It downloads every mod and config
   straight from CurseForge/Modrinth into your `.minecraft` folder.
4. In TLauncher settings, set memory to **6–8 GB**.
5. Enter your username (ask the admin to whitelist it — the name IS your identity on an
   offline-mode server, so never change it), then join the server.

## Updating

Run `update.bat` again. That's it — it syncs you to exactly the version the server runs.

## Notes

- If TLauncher uses a custom game directory, run: `update.bat "C:\path\to\that\folder"`.
- Skins: the pack ships an offline-skin mod, so your skin works without a premium account.
- Voice chat: the pack includes Simple Voice Chat — plug in a mic and press `V` in game.
